Task Description

Need to analyze the data files and create a schemas (in the form of SQL statements to create tables) in accordance with the tasks:
1. A snowflake schema for the BI analyst to work with data on Nobel laureates (including descriptive information on countries and cities).
2. A dataset's schema for the Data Scientist; a dataset should contain data on Nobel laureates (including descriptive information on countries and cities).

Original data files are stored in Kaggle:
https://www.kaggle.com/timoboz/country-data
https://www.kaggle.com/fernandol/countries-of-the-world
https://www.kaggle.com/max-mind/world-cities-database
https://www.kaggle.com/nobelfoundation/nobel-laureates
